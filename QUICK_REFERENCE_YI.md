## 2. אריגינאל און דיסקאונט פרייז אויפן ביל ✅

**נייע פעלדער אין OrderItem:**
- `original_price` - דער אמת'ע פרייז פאר דיסקאונט
- `discount_percentage` - וויפיל פראצענט דיסקאונט
- `unit_price` - דער פרייז נאך דיסקאונט

**ווי אזוי עס ווייזט אויס:**
- אריגינאל פרייז מיט א שטריך דורך (ווען עס איז א דיסקאונט)
- דיסקאונט פרייז אין גרין
- דיסקאונט באדזש (פראצענט אדער סכום)

**ווי אזוי צו נוצן:**
ווען דו מאכסט אן ארדער איטעם, לייג אריין:
- `unit_price` - דער פרייז וואס דער קאסטאמער באצאלט
- `original_price` - דער אריגינאל פרייז
- `discount_percentage` - וויפיל פראצענט (אפציאנעל)

**מיגרייטשאן:**
```powershell
python manage.py migrate orders
```

## 3. דאקומענט מענעדזמענט סיסטעם ✅

**וואס מען קען אריינלייגן:**
- ינוואויסעס (פון סופלייערס און פאר קאסטאמערס)
- ספרים ליסטן פון סופלייערס
- עדס (אדווערטייזמענטס)
- פערטשעס ארדערס
- רעסיטס
- קאנטראקטן
- עמעילס און קארעספאנדענץ
- אנדערע דאקומענטן

**ווו דאס ווערט געהאלטן:**
`media/documents/{טיפ}/{יאר}/{חודש}/`

**ווי אזוי צו נוצן:**
1. גיי צו `/documents/` - זען אלע דאקומענטן
2. גיי צו `/documents/upload/` - ארויפלייגן א נייעם דאקומענט
3. אדער אין Django Admin → Documents

**פיטשערס:**
- זוכן לויט טיפ, דאטע, טאגס
- לינקען דאקומענטן צו ארדערס, סופלייערס, ספרים
- רעפערענץ נומערן (ינוואויס #, PO #)
- נאטיצן און טאגס

**מיגרייטשאן:**
```powershell
python manage.py migrate documents
```

## 4. מתנה קארטלעך סיסטעם ✅

**וואס עס טוט:**
- אויטאמאטיש שאפן קארטל נומערן (GC-XXXX-XXXX-XXXX)
- האלטן חשבון פון באלאנס
- טראקן אלע טרענזעקשאנס
- לינקען צו קאסטאמערס
- סטאטוס: אקטיוו, גענוצט, אויסגעגאנגען, אנולירט

**ווי אזוי צו שאפן א קארטל:**
1. Django Admin → Gift Cards → Add Gift Card
2. לייג אריין initial balance (סכום)
3. קען לינקען צו א קאסטאמער (אפציאנעל)
4. קארטל נומער ווערט אויטאמאטיש געשאפן
5. סעיוו

**ווי אזוי צו נוצן א קארטל פאר א צאלונג:**
```python
gift_card = GiftCard.objects.get(card_number='GC-1234-5678-9012')
is_valid, message = gift_card.is_valid()  # טשעק אויב גילטיק

if is_valid:
    amount = gift_card.deduct(
        amount=50.00,
        description="ארדער #12345"
    )
```

**ווי אזוי צו לאדן אויף א קארטל:**
```python
gift_card.add_balance(
    amount=100.00,
    description="רילאוד קארטל"
)
```

**מיגרייטשאן:**
```powershell
python manage.py migrate customers
```

## 5. צאלונג סינכראניזאציע ✅

**פראבלעם געפיקסט:**
- ארדערס האבן געזאגט "נישט באצאלט" כאטש דער קאסטאמער האט באצאלט
- צאלונגען האבן נישט געצויגן צווישן קאסטאמער און ארדער

**נייער מאדעל: CustomerPayment**

**וואס עס טראקט:**
- אלע צאלונגען פון קאסטאמערס
- לינק צו ספעציפישע ארדערס
- צאלונג מעטאדן (קעש, קרעדיט קארד, וכו')
- רעפערענץ נומערן (טשעק #, טרענזעקשאן ID)

**ווי אזוי צו רעקארדירן א צאלונג:**
```python
payment = CustomerPayment.objects.create(
    customer=customer,
    amount=100.00,
    payment_method='cash',
    order=order,
    notes='צאלונג פאר ספרים'
)

# אפליקירן די צאלונג צום ארדער
payment.apply_to_order()
```

**אין Admin:**
- Django Admin → Customer Payments
- זעה אלע צאלונגען
- באלק אקשאן: "Apply payments to linked orders"

**מיגרייטשאן:**
```powershell
python manage.py migrate customers
```
